## 🚀 시작하기
### Installation
```bash
# 저장소 클론
git clone https://github.com/boostcampwm-2024/web20-camon.git

# 의존성 설치
pnpm install

# client
cd apps/client
pnpm dev

# api / media / chat 서버
cd apps/{SERVER_NAME}
pnpm start
```
